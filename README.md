# Reflecta-MirrorV1
Smart Mirror using a Raspberry Pi 4 and the Magic Mirror software. I will completely customize the Mirror to my liking, and will be looking for cool ideas/features to add to the mirror. This project is part of HackClub Blueprint! 

Parts list: 
Raspberry Pi Model B 2GB 
USB-C Power Supply 5.1v
MicroSD 32GB 
Micro-HDMI to HDMI cable 
RPi FPC Camera (B) 5MP mini
One-way/two-way mirror (can be acrylic as well)
Monitor/ TV
Frame to hold all parts
