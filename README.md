# Reflecta Mirror V1

A Raspberry Pi 4-powered smart mirror for my bedroom, 
running MagicMirror with custom modules and a built-in 
camera module.

## Software

This project uses MagicMirror²:
https://github.com/MichMich/MagicMirror

It is included in the /MagicMirror folder.

Built as part of [Hack Club Blueprint](https://blueprint.hackclub.com/projects/13749)

---

## What it does

This is a smart mirror that displays live information 
like the time, weather, and calendar through a two-way 
acrylic mirror panel. It also has a built-in camera 
module for hand tracking and photos. The whole thing 
runs on a Raspberry Pi 4 mounted inside a custom 
wooden frame I built myself.

## Why I made it

I wanted a project that would make my room less boring 
and actually teach me something real. I have made a 
game in Godot before at Campfire and attempted a macropad through 
Hackclub, but I wanted to build something that combines 
hardware, software, and design all in one. This felt 
like the right challenge.

## How to use it

1. Power on the mirror using the USB-C switch on the back
2. The MagicMirror interface will boot automatically
3. Widgets display live data, including time, weather, and your Google Calendar
4. Use your phone as a remote to switch between modules## Parts List

## Build Plan

1. Source and prepare the used monitor
2. Set up Raspberry Pi 4 with Raspberry Pi OS
3. Install and configure the MagicMirror software on the Pi
4. Add custom modules to your liking! (weather, clock, calendar, camera)
5. Cut the acrylic mirror panel to your choosing
6. Secure monitor behind panel
7. Assemble the frame with the monitor and mirror panel
8. Mount Pi and cables inside the frame
9. Wall mount and final cleanup
